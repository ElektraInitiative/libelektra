kdb-umount(1) - Unmount a file from the key database
====================================================

## SYNOPSIS

`kdb umount <name>`

## DESCRIPTION

Unmount backend from key database.

## OPTIONS

- `-H`, `--help`:
  Show the man page.
- `-V`, `--version`:
  Print version info.
- `-p`, `--profile`=<profile>:
  Use a different kdb profile.
- `-v`, `--verbose`:
  Show which mountpoints were considered.
- `-C`, `--color`=[when]:
  Print never/auto(default)/always colored output.
